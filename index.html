TableFixer Script

Converts an HTML table into more responsive elements while maintaining the structure. The button below will run the script. Refresh the page to see the original again.

Apply TableFix

During my internship at RVCC, the college decided to make a new and fully responsive web site in a new CMS. The task of migrating hundreds, if not thousands, of old pages fell to a single webmaster and two part-time student interns. To make matters worse, most old pages were made by non-technical faculty members, and many used tables for layout.

The temporary solution I came up with was a bit of javascript that transformed tables into divs with Bootstrap classes. It doesn't produce very good HTML and the results can be unpredictable, but it does work. I've adapted this example to work without Bootstrap.

At the time of writng RVCC is still using their old site.
