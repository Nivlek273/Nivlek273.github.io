TableFixer Script

Converts an HTML table into more responsive elements while maintaining the structure. The button below will run the script. Refresh the page to see the original again.

Apply TableFix

During my internship at RVCC the college decided to make their website responsive. Unfortunately, hundreds of pages on the site used tables for layout. We knew these pages would eventually have to be redesigned or replaced, but we didn't have the time or manpower to do it within a reasoble time frame.

The temporary solution I came up with was a javascript plugin that transformed tables into divs with Bootstrap classes. The 'no-tableFix' class can be added to tables that should be skipped. The script doesn't produce very good HTML and the results can be unpredictable, but it does work. I've adapted this example to work without Bootstrap.
